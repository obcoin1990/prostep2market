# UI Components

## Buttons
- **Primary** background: var(--brand-red); text: var(--white); border-radius: var(--radius-md)  
- **Secondary** background: transparent; border: 1px solid var(--brand-black); text: var(--brand-black)  
- **Success** background: var(--brand-green); text: var(--white)

## Cards
- Background: var(--white) or var(--muted-gray) for subtle elevation  
- Border-radius: var(--radius-md)  
- Shadow: subtle (e.g., 0 6px 18px rgba(11,11,11,0.06))

## Alerts & Badges
- **Warning**: muted red background with brand red border  
- **Success**: muted green background with brand green border  
- Use icons + color for accessibility

## Dashboard Widgets
- **Edge Score Card**: large numeric, small trend sparkline, micro tips in brand green  
- **Emotional Risk Meter**: gradient from green to red with numeric index and short action text
