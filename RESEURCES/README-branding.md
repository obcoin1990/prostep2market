# ProStep2Market Branding Resources

This folder contains branding and style resources for ProStep2Market.

Files:
- brand-guidelines.md
- brand-assets.md
- style-tokens.md
- components.md
- marketing-copy.md
- home-hero.md

Usage:
- Drop SVG/PNG logo files into assets/logo and update paths in frontmatter.
- Use style tokens in Tailwind config and global CSS.
- Follow logo usage rules when placing the mark in UI and marketing materials.
