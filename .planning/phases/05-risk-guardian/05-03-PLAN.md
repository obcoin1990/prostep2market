## Supabase Realtime Subscription (from RESEARCH.md)

```typescript
supabase
  .channel(`alerts:${userId}`)
  .on('postgres_changes', {
    event: 'INSERT',
    schema: 'public',
    table: 'alerts',
    filter: `user_id=eq.${userId}`
  }, (payload) => {
    showAlertNotification(payload.new)
  })
  .subscribe()
```

  <action>
Create `src/hooks/useAlertSubscription.ts` implementing Supabase Realtime subscription.

Export `useAlertSubscription(userId: string)`:
1. On mount: subscribe to `alerts:${userId}` channel with `postgres_changes` INSERT filter
2. On INSERT event: call `alertStore.getState().addAlert(payload.new as Alert)` then call `showAlert(payload.new as Alert)`
3. Implement connection health check every 5 minutes (setInterval) — if channel state !== 'joined', set `reconnecting: true` and re-subscribe
4. Re-subscribe on `visibilitychange` (document visibility API)
5. On unmount: `supabase.removeChannel(channel)`

Also export a `showAlert(alert: Alert)` function that calls Sonner toast:
- critical: `toast.error(alert.message, { id: alert.id, duration: 0, action: {...}, cancel: {...} })`
- warning: `toast.warning(alert.message, { id: alert.id, duration: 8000 })`
- info: `toast.info(alert.message, { id: alert.id, duration: 5000 })`

The action button links to alert detail/history. The cancel button dismisses.

Import `alertStore` from `~/lib/risk-guardian/alertStore`.
Import Supabase from `~/lib/supabase/client`.
